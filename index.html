Terminal Commands

General Information about Terminal

Listen

  • ls - Liste anzeigen
  • ls()-1 - Liste untereinander anzeigen
  • ls()-l - Leserechte anzeigen
  • ls -a - versteckte Dateien in Liste anzeigen

Change Directory

  • cd().. - Change Directory - eine Ebene in Ordnerstruktur zurückgehen
  • cd NAME_ORDNER TABTASTE - Autovervollständigung des ordner Namens
  • clear - Fenster löschen
  • cd()Tilde - in Überordner gelangen

Make Directory

  • mkdir - einen neuen Ordner erstellen
  • mkdir NAME_ORDNER/NAME_ORDNER - einen Ordner in einen anderen Ordner legen
  • rmdir=remove directory - Ordner löschen

Dateien

  • touch - Datei erstellen
  • mv - Datei verschieben
  • cp - Datei kopieren
  • rm - remove file - Datei löschen
  • searchforfiles - suche nach Dateien

Terminal

  • exit - Terminal Fenster schließen
  • # - Kommentare in Terminal schreiben
  • whoami - zeigt aktuellen user an
  • sudo - wenn man etwas nicht ausführen kann
  • date - aktuelles Datum anzeigen
  • nano - öffnen von Text editor nano
  • history - zeigt alle Befehle an, die ich angegeben habe
  • whatis - Kurzinfo zum angegebenen Befehl
